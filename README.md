OpenCV的JavaScript绑定。

## 如何使用

异步加载opencv.js:

```html
<script async src="opencv.js" onload="onOpenCVReady();" type="text/javascript"></script>

<script type="text/javascript">

    function onOpenCVReady() 
    {
        // do something...
    }
    
</script>
```
